Cursor OPEN Statement


#### 15.6.6.4 Cursor OPEN Statement

```sql
OPEN cursor_name
```

This statement opens a previously declared cursor. For an
example, see [Section 15.6.6, “Cursors”](https://dev.mysql.com/doc/refman/8.0/en/cursors.html "15.6.6 Cursors").
