## BetterDocs Category Box

Display your BetterDocs knowledgebase in stunning box style on your Elementor site using EA BetterDocs Category Box.  You will get the full freedom to customize the layout section and style it to make it interactive.

[Documentation](https://essential-addons.com/elementor/docs/betterdocs-category-box/)

![Betterdocs Category Box 101](https://essential-addons.com/elementor/wp-content/uploads/2020/01/Video-Icon-Update-1.svg)

### Configure BetterDocs Category Box Layout

Setup the ‘Query’ settings to choose which document will display on the website using the include or exclude option with others. Also, pick your preferred layout from ‘Layout’ settings, and style it in advance to make your knowledgebase standout.
