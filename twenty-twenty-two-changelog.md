## [Version 2.0](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_2.0)

Released: April 15, 2025 with [WordPress 6.8](https://wordpress.org/documentation/wordpress-version/version-6-8/)

- Fix coding standards issues in `functions.php`. [#62648](https://core.trac.wordpress.org/ticket/62648)

## [Version 1.9](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.9)

Released: November 12th, 2024 with [WordPress 6.7](https://wordpress.org/documentation/wordpress-version/version-6-7/)

- Add instructions for updating the Comments block from the legacy version to the newer version introduced in WordPress 6.1. ( [#56496](https://core.trac.wordpress.org/ticket/56496))
- Fix custom Search block border by setting it to zero width instead of removing all borders. ( [#60669](https://core.trac.wordpress.org/ticket/60669))
- Update schema version in style variation files to ensure that settings and styles are properly handled by code editors that support schema. ( [#61789](https://core.trac.wordpress.org/ticket/61789))
- Add contributors to copyright notices. ( [#61943](https://core.trac.wordpress.org/ticket/61943))

## [Version 1.8](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.8)

Released: July 16th, 2024 with [WordPress 6.6](https://wordpress.org/documentation/wordpress-version/version-6-6/)

- Wrap Navigation block in a Row block within the header template part. ( [#60724](https://core.trac.wordpress.org/ticket/60724))
- Set `theme.json` schema version to 6.5, the last schema for version 2, instead of the development version at `trunk`. ( [#61342](https://core.trac.wordpress.org/ticket/61342))

## [Version 1.7](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.7)

Released: April 2nd, 2024 with [WordPress 6.5](https://wordpress.org/documentation/wordpress-version/version-6-5/)

- Prefix the `pages` block pattern category name. ( [#59839](https://core.trac.wordpress.org/ticket/59839))

## [Version 1.6](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.6)

Released: November 7, 2023 with [WordPress 6.4](https://wordpress.org/documentation/wordpress-version/version-6-4/)

- [#59256](https://core.trac.wordpress.org/ticket/59256) – Add missing dimension attributes to images in Twenty Twenty-Two patterns.

## [Version 1.5](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.5)

Released: August 8, 2023 with [WordPress 6.3](https://wordpress.org/documentation/wordpress-version/version-6-3/)

- Improve the layout of the comment section by applying a bottom margin to top-level comments ( [#58653](https://core.trac.wordpress.org/ticket/58653))
- Add missing subject tags and feature tags ( [#58437](https://core.trac.wordpress.org/ticket/58437))

## [Version 1.4](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.4)

Released: March 29, 2023 with [WordPress 6.2](https://wordpress.org/news/2023/03/dolphy/)

- [#57026](https://core.trac.wordpress.org/ticket/57026) – Replace “Full site editing” with “Site Editor”.
- [#56720](https://core.trac.wordpress.org/ticket/57026) – Remove color reference from pattern names.
- [#57560](https://core.trac.wordpress.org/ticket/57560) – margin and padding issue in comments section
- [#57726](https://core.trac.wordpress.org/ticket/57726) – Use the new `/documentation/` URLs for HelpHub links in Bundled Themes.

## [Version 1.3](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.3)

Released: November 1, 2022 with [WordPress 6.1](https://wordpress.org/news/2022/11/misha/)

- [#55172](https://core.trac.wordpress.org/ticket/55172) – Add “Accessibility Ready” tag to theme in repository
- [#55697](https://core.trac.wordpress.org/ticket/55697) – Dark mode not applied to sub menu in navigation block
- [#55772](https://core.trac.wordpress.org/ticket/55772) – Typo in docblock, in footer.php

## [Version 1.2](https://wordpress.org/documentation/article/twenty-twenty-two-changelog/\#Version_1.2)
